Interactive mode

Launch the TUI for browsing and managing secrets:

$ zvault

You'll be prompted for your master password, then presented with an interactive interface for navigating entries, searching, and copying secrets to the clipboard.

CLI commands

Retrieve a secret

$ zvault get <path>

Prints the value of the entry at <path> to stdout. Useful for scripting and piping into other commands.

Store a secret

$ zvault set <path>

Prompts for the secret value and stores it encrypted at the given path. If the path already exists, the entry is overwritten.

Search entries

$ zvault search <query>

Searches entry names, tags, and folder paths. Returns matching entries without revealing secret values.
