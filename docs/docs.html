zvault docs
These commands are planned — zvault is not yet released. This page documents the intended CLI interface.
zvault init

Initialize a new encrypted vault. Creates the vault directory and sets your master password.

$ zvault init

You'll be prompted to create a master password. The password is used with Argon2id to derive your encryption key. The vault stores all data locally in ~/.local/share/zvault/.

zvault store

Store a new secret in the vault. Prompts for the secret value and encrypts it with age encryption.

$ zvault store <name>

The name is the key used to retrieve the secret later. If a secret with that name already exists, it will be overwritten.

zvault get

Retrieve a secret from the vault. Prints the decrypted value to stdout.

$ zvault get <name>

Useful for scripting and piping into other commands. Requires master password authentication.

zvault list

List all stored secrets. Shows names without revealing values.

$ zvault list
zvault delete

Permanently remove a secret from the vault.

$ zvault delete <name>

This action is irreversible. The encrypted data is securely erased from disk.
