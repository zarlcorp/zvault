zvault
encrypted local storage for secrets, keys, notes. your data, your machine, your keys.
features
  • encrypted local vault with age encryption
  • key management and secret storage
  • secure notes with full-text search
  • interactive TUI + scriptable CLI
  • task tracking with priorities and due dates
  • export to markdown, JSON, CSV
  • single binary, no cloud, no accounts
install

via homebrew:

brew install zarlcorp/tap/zvault

via go:

go install github.com/zarlcorp/zvault/cmd/zvault@latest
quickstart

launch the interactive TUI:

zvault

or use the CLI directly:

# store a password
zvault secret store -t password -n github

# add a task
zvault task add -p h -d tomorrow "deploy to prod"

# list secrets
zvault secret list

# search
zvault secret search github

see the full documentation for all commands.
